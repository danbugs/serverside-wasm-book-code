# `hello_world_wasi02`

## How to re-create this example?

```bash
cargo component new --bin hello_world_wasi02
```

## How to run?

```bash
wasmtime ../../target/wasm32-wasip1/release/hello_world_wasi02.wasm
Hello, world!
```
